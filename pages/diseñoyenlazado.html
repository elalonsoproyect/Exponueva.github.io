Most used APIs

Google Maps: Allows us to embed maps in our own site using JavaScript, being able to use and manipulate them as we need.

Youtube: We can use it to create websites and also for desktop apps so that they can obtain information from both users and videos.

Amazon S3: Allows you to save files in an infrastructure provided by Amazon at a price lower than what it would cost us to create our own service network.
